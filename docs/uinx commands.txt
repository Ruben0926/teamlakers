Uinx Commands
=============
pwd - print (show) working directory
cd - change directory (.. parent directory)
mkdir - make/create directory
nano - starts Nano text editor
text editor: Nano, Pico, Emacs, Vi, Vim
cat - shows contents of file
ls - list directory (ls -l, long list)
rm - remove directory or file
clear - clear screen
mv - move or rename
cp - copy
touch - create or update a file
